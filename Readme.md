# CP-Templates

A collection of optimized data structures, algorithms, and boilerplate templates for fast access and retrieval.

## 🚀 Repository Structure

| Directory | Description |
| :--- | :--- |
| **`BinaryLifting_And_LCA`** | Lowest Common Ancestor using binary lifting techniques. |
| **`DSU`** | Disjoint Set Union (Union-Find) with path compression and rank/size optimization. |
| **`Input`** | Fast I/O templates to minimize execution overhead. |
| **`Sieve`** | Prime factorization and Sieve of Eratosthenes implementations. |
| **`Trie`** | Prefix tree structures for efficient string and bitwise operations. |
| **`nCr`** | Combinatorics, modular inverse, and nCr computations. |
| **`segmentTree`** | Range query and point/range update data structures. |

## 🛠️ Environment & Language
* **Primary Language:** Python
* **Goal:** Fast and remote access.
